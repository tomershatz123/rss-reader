# RSS Reader

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 7.1.3.

## Build

1. Download the repository.
2. Run `npm install` 
3. Run `ng serve`.
4. Navigate to `http://localhost:4200/`. The app will automatically reload.
